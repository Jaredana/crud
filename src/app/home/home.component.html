What I learned during this project

This project took me quite some time considering I had never touched much web app development prior to making this besides a touch of Python's Django
All in all, this project took me about 3-4 weeks of development time with around 10hrs of dedicated development time per week

I'll go ahead and do my best to break down my forray into these technologies and describe my thought and issues with each one

Angular
Components:

One of the things I found the most interesting about development with angular is the use of components to handle views.
This was one of the most stark contrasts between development in Django and development in Angular for me. I particularly liked Angulars design philopsophy
a bit more because it felt more compartmentalized and abstracted. My data fetching belonged to my service, my component held that data and displayed it in the view
to me, this is an extremely simple and powerful way to develop apps like this because it keeps even developers with as little experience as me from writing complete
spaghetti code.
